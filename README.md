📊 Customer Behavior Analysis & Dashboard

📌 About This Project

This project is focused on understanding customer behavior using data analysis, SQL, and visualization. The goal was to explore how customers interact with a business, identify patterns in their purchasing behavior, and present those insights in a clear and interactive dashboard.

I worked on this project end-to-end — from cleaning raw data and writing SQL queries to building a dashboard that highlights key business metrics.

🧩 Project Workflow

This project was completed in three main stages:

1. Data Cleaning & Preparation (Python)
Cleaned and preprocessed raw data using Pandas
Handled missing values and corrected data types
Prepared the dataset for analysis
2. Data Analysis (SQL)
Wrote SQL queries to solve real business problems
Analyzed customer behavior, revenue trends, and product performance
Extracted insights such as:
Revenue by gender
Impact of discounts on purchases
Top-rated and most purchased products
Customer segmentation (New, Returning, Loyal)
Subscription vs non-subscription behavior
Revenue contribution by age group

👉 Example: SQL queries used for analysis

4. Dashboard Creation (Visualization)
Built an interactive dashboard to present insights
Focused on making data easy to understand for business decisions

📊 Dashboard Overview

The dashboard provides a quick snapshot of key metrics and trends:

Total Customers: 627
Average Purchase Amount: $60.73
Average Rating: 3.77
Key Visuals:
Customer distribution by subscription status
Sales and purchase trends by category
Revenue and sales across different age groups

📈 Key Insights

Most customers are non-subscribers, indicating an opportunity for subscription growth
Certain categories (like Clothing) drive the majority of sales
Older age groups contribute more to overall revenue
Customers with higher purchase frequency tend to be more valuable
Discounts influence purchasing behavior but don’t always guarantee higher spending

💡 Why This Project Matters

This project shows how raw data can be transformed into meaningful insights that help businesses:

Understand their customers better
Identify high-value customer segments
Improve marketing and retention strategies
Make data-driven decisions
